# Odor, Drying, and Storage

## Odor control

- Smell is usually manageable in veg and much stronger in flower and drying.
- Drying can smell stronger than the grow itself.
- If odor matters, use negative pressure plus inline fan and carbon filter.
- If odor does not matter, skip the carbon filter and preserve budget for better light or more reliable hardware.

## Drying

- Dry in the tent if it gives you the best environmental control.
- Keep lights off.
- Run exhaust and filtration.
- Use gentle circulation, not direct fan blast on the material.
- Target 18-21 C and 55-60% RH.

## Storage

- Humidity packs are for curing and storage, not the initial dry.
- A 62% pack does not create mold by itself if the material was dried correctly before jarring.
- Mold risk rises if jar RH stays above roughly `65%`, storage is warm, or material was jarred too wet.
- Rehydration can improve texture, but it does not restore lost terpenes or reverse age-related degradation.
- Check rehydration after about `24 hours`, then again over `3-7 days`.
- Long storage still reduces terpene intensity and gradually changes cannabinoid profile.

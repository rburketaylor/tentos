# Environment Targets

## Main targets

| Stage | Temp | RH |
|---|---|---|
| Seedling | 22-26 C | 65-75% |
| Vegetative | 22-28 C day / 18-22 C night | 55-70% |
| Early flower | 20-26 C | 45-55% |
| Late flower | 20-24 C | 35-45% |
| Drying | 18-21 C | 55-60% |

## Current practical guidance

- Light and airflow quality matter more than adding automation early.
- The LED often provides enough heat while lights are on in a cool room, but not always at lights-off.
- If the room is around `16-20 C`, a small tent with LED lighting often stays acceptable while lights are on.
- Below about `16 C`, growth tends to slow and supplemental heat becomes more likely to matter.
- Start with temp/RH monitoring before buying extra heating or humidity hardware.
- Compute VPD later if needed; basic temp/RH targets are enough to get the system stable first.

## Equipment order

1. Light
2. Exhaust and circulation
3. Monitoring
4. Humidity control
5. Supplemental heat only if real measurements prove it is needed
